goog.userAgent.adobeReader.SILENT_PRINT :
boolean
On certain combinations of platform/browser/plugin, a print dialog can be shown for PDF files without a download dialog or making the PDF visible to the user, by loading the PDF into a hidden iframe. Currently this variable is true if Adobe Reader version 6 or later is detected on Windows.
