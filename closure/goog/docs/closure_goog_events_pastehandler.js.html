pastehandler.js
Provides a 'paste' event detector that works consistently across different browsers. IE5, IE6, IE7, Safari3.0 and FF3.0 all fire 'paste' events on textareas. FF2 doesn't. This class uses 'paste' events when they are available and uses heuristics to detect the 'paste' event when they are not available. Known issue: will not detect paste events in FF2 if you pasted exactly the same existing text. Known issue: Opera + Mac doesn't work properly because of the meta key. We can probably fix that. goog.events.PasteHandler
A paste event detector. Gets an element as parameter and fires goog.events.PasteHandler.EventType.PASTE events when text is pasted in the element. Uses heuristics to detect paste events in FF2. See more details of the heuristic on #handleEvent_.
