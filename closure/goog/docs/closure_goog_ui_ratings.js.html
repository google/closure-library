ratings.js
A base ratings widget that allows the user to select a rating, like "star video" in Google Video. This fires a "change" event when the user selects a rating. Keyboard: ESC = Clear (if supported) Home = 1 star End = Full rating Left arrow = Decrease rating Right arrow = Increase rating 0 = Clear (if supported) 1 - 9 = nth star
