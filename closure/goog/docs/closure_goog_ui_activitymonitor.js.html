activitymonitor.js
Activity Monitor. Fires throttled events when a user interacts with the specified document. This class also exposes the amount of time since the last user event. If you would prefer to get BECOME_ACTIVE and BECOME_IDLE events when the user changes states, then you should use the IdleTimer class instead.
