testrunner.js
The test runner is a singleton object that is used to execute a goog.testing.TestCases, display the results, and expose the results to Selenium for automation. If a TestCase hasn't been registered with the runner by the time window.onload occurs, the testRunner will try to auto- discover JsUnit style test pages. The hooks for selenium are :- - Boolean G_testRunner.isFinished() - Boolean G_testRunner.isSuccess() - String G_testRunner.getReport() - number G_testRunner.getRunTime() Testing code should not have dependencies outside of goog.testing so as to reduce the chance of masking missing dependencies.
