idletimer.js
Idle Timer. Keeps track of transitions between active and idle. This class is built on top of ActivityMonitor. Whenever an active user becomes idle, this class dispatches a BECOME_IDLE event. Whenever an idle user becomes active, this class dispatches a BECOME_ACTIVE event. The amount of inactive time it takes for a user to be considered idle is specified by the client, and different instances of this class can all use different thresholds.
