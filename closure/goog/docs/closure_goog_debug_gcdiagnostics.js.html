goog.debug.GcDiagnostics_
Class used for singleton goog.debug.GcDiagnostics. Used to hook into the L2 ActiveX controller to profile garbage collection information in IE. Can be used in combination with tracers (goog.debug.Trace), to provide object allocation counts from within the tracers or used alone by invoking start and stop. See http://go/l2binary for the install. TODO: Move the L2 installer somewhere more general.
