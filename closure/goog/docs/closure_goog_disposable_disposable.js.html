goog.Disposable
Class that provides the basic implementation for disposable objects. If your class holds one or more references to COM objects, DOM nodes, or other disposable objects, it should extend this class or implement the disposable interface.

goog.dispose(obj)
Calls dispose on the argument if it supports it. If obj is not an object with a dispose() method, this is a no-op.
Arguments:
obj :
*
The object to dispose of.
