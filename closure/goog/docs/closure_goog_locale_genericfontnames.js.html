genericfontnames.js
Functions to list locale-specific font list and generic name. Generic name used for a font family would be locale dependant. For example, for 'zh'(Chinese) users, the name for Serif family would be in Chinese. Further documentation at: http://go/genericfontnames.

goog.locale.genericFontNames.getList(locale) ⇒
Array.<Object>
Gets the list of fonts and their generic names for the given locale.
Arguments:
locale :
string
The locale for which font lists and font family names to be produced. The expected locale id is as described in http://wiki/Main/IIISynonyms in all lowercase for easy matching. Smallest possible id is expected. Examples: 'zh', 'zh-tw', 'iw' instead of 'zh-CN', 'zh-Hant-TW', 'he'.
Returns: 
Array.<Object>
  List of objects with generic name as 'caption' and corresponding font name lists as 'value' property.
