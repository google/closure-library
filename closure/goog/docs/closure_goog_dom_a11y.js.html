a11y.js
Utilities for adding, removing and setting ARIA roles as defined by W3C ARIA Working Draft: http://www.w3.org/TR/2008/WD-wai-aria-20080806/ At this time Firefox 3.0 is the only browser that has good support for ARIA; IE8 (beta) has partial support, while Opera and WebKit implementations are forthcoming. An application could call goog.dom.a11y.setNoBrowserCheck with true to turn on ARIA support for all browsers.
