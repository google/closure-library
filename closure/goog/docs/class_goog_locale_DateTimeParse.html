markAbutStart_()
Identify the start of an abutting numeric fields' run. Taking pattern "HHmmss" as an example. It will try to parse 2/2/2 characters of the input text, then if that fails, 1/2/2. We only adjust the width of the leftmost field; the others remain fixed. This allows "123456" => 12:34:56, but "12345" => 1:23:45. Likewise, for the pattern "yyyyMMdd" we try 4/2/2, 3/2/2, 2/2/2, and finally 1/2/2. The first field of connected numeric fields will be marked as abutStart, its width can be reduced to accomodate others.
