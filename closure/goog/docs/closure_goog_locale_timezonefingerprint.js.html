timezonefingerprint.js
Data for time zone detection. The following code was generated by the timezone_detect.py script in: http://go/i18n_tools which uses following files in this directory: http://go/timezone_data Files: olson2fingerprint.txt, country2olsons.txt, popular_olsons.txt After automatic generation, we added some manual editing. Projecting on future changes, it is very unlikely that we will need to change the time zone ID groups. Most of the further modifications will be about relative time zone order in each time zone group. The easiest way to do that is to modify this code directly, and that's what we decide to do.
