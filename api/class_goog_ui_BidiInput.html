getDirection()code »

Returns the direction of the input element.

Returns

Return 'rtl' for right-to-left text, 'ltr' for left-to-right text, or null if the value itself is not enough to determine directionality (e.g. an empty value), and the direction is inherited from a parent element (typically the body element).
