convertToAsyncTestObj(original)

Wraps an object's methods by passing in a Queue that is based on the JSTD async API. The queue exposes a promise that resolves when the queue completes. This promise can be used in JsUnit tests.

Parameters
original!Object

The original JSTD test object. The object should contain methods such as testXyz or setUp.

Returns
!Object

A object that has all test methods wrapped in a fake testing queue.
