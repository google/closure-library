16/**
17 * @fileoverview Constants used by the WebGL rendering, including all of the
18 * constants used from the WebGL context. For example, instead of using
19 * context.ARRAY_BUFFER, your code can use
20 * goog.webgl.ARRAY_BUFFER. The benefits for doing this include allowing
21 * the compiler to optimize your code so that the compiled code does not have to
22 * contain large strings to reference these properties, and reducing runtime
23 * property access.
24 *
25 * Values are taken from the WebGL Spec:
26 * https://www.khronos.org/registry/webgl/specs/1.0/#WEBGLRENDERINGCONTEXT
27 */
